# OpenReturn

Open protocol and tooling for e-commerce returns and exchanges.

OpenReturn defines an open, machine-readable standard for the return lifecycle that any e-commerce platform, headless commerce tool, or AI agent can integrate with. It includes a self-hostable reference portal (Next.js), modular carrier and platform adapters using retailer-owned API keys, so no intermediary touches the retailer's data or takes a margin on their shipments.

Designed to be compatible with [UCP](https://ucp.dev), [MCP](https://modelcontextprotocol.io) and [A2A](https://google.github.io/A2A/).
